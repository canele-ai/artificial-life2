# Problem Specification

> Generated by Phase 1.5 Problem Refinement panel (3 proposers + reviewer).
> Supersedes aspirational Phase 1 decisions. Ground truth for Phase 2.

## Refined Research Question

Discover simulation parameters (in Lenia and Flow-Lenia) that produce rollouts
whose temporal FM-embedding trajectories match ordered life-cycle prompt
sequences — scored by the worst of three held-out image foundation models
(CLIP ViT-B/32, DINOv2-base, SigLIP ViT-B/16), gated on a permutation-order
check and existence-of-pattern check, measured against a pinned ASAL
Sep-CMA-ES zero-anchor baseline.

Single goal: **beat the pinned ASAL baseline on the held-out split by METRIC
> 0 with seed-paired Wilcoxon p < 0.05.**

## Metric

- **Name:** `LCF-robust@heldout` (Life-Cycle Fidelity, worst-of-FMs, held-out)

- **Formula:**
  ```
  Inputs per (solution_params θ, seed s ∈ S_test, prompt sequence p ∈ P_test, FM f ∈ F):
    frames  = rollout(substrate, θ, s, T=256)                        # [T, H, W, 3] uint8
    picks   = frames[[0, 63, 127, 191, 255]]                          # L=5 ordered frames
    Z_img   = f.encode_image(picks)                                   # [L, d]
    Z_txt   = f.encode_text(p)                                        # [L, d]   (L matched prompts)
    A       = softmax(Z_img @ Z_txt.T / 0.07, axis=1)                 # [L, L]
    OT      = mean_i A[i,i] − mean_{i≠j} A[i,j]                       # diagonal dominance
    PG      = OT − max over Π ∈ {reverse, 10 fixed shuffles} of OT(Π·p)  # permutation gap

  Existence gate E(θ, s):
    E = 1 iff final_mass(θ,s) ∈ [0.1×, 10×] initial_mass
            AND variance(last_32_frames) > 1e-4
        else 0

  PG_eff(θ, s, p, f) = E(θ, s) · max(0, PG(θ, s, p, f))

  LCF(θ, f) = median over p ∈ P_test  [ mean over s ∈ S_test of PG_eff ]
  LCF_anchor = LCF computed on pinned ASAL baseline at campaign init (frozen)

  METRIC(θ) = min over f ∈ F of LCF(θ, f)  −  min over f of LCF_anchor(f)
  ```
  F = {`clip_vitb32`, `dinov2_vitb14`, `siglip_vitb16`} (SigLIP-1; SigLIP-2
  deferred unless Flax port becomes available).
  |S_test| = 16 (held out from the 32-seed pool). |P_test| = 10 sequences × 5
  prompts each (held out from the 20-sequence authored bank).

- **Direction:** maximize. Expected range ≈ [−0.05, +0.20]. `> 0` means beats
  ASAL baseline.

- **Provenance:** synthesizes ASAL's softmax-target metric (Kumar et al. 2024,
  *Automating the Search for Artificial Life with Foundation Models*,
  [arXiv:2412.17799](https://arxiv.org/abs/2412.17799), `asal_metrics.py:calc_supervised_target_softmax_score`),
  with permutation-gap regularization (novel to this campaign), cross-FM
  worst-of aggregation (inspired by ensemble-judge literature, novel to ALife
  FM search), and Mouret-Clune-style pinned-baseline anchoring
  ([arXiv:1504.04909](https://arxiv.org/abs/1504.04909)).

- **Target:** no hard target. Pass bar = METRIC > 0 at Wilcoxon p < 0.05.
  Stretch: METRIC > 0.05.

- **Known pitfalls (and guards):**
  - *CLIP-only overfit:* guarded by worst-of-three FMs.
  - *Unordered match gaming:* guarded by permutation gap PG.
  - *Chaotic high-variance rollouts:* guarded by softmax-diagonal-dominance OT.
  - *Dead/exploded patterns:* guarded by existence gate E.
  - *Prompt overfit:* guarded by held-out P_test disjoint from P_train.
  - *Seed lottery:* guarded by held-out S_test + Wilcoxon over seeds.
  - *Scorer-as-oracle:* `search()` may request one `search_fm`; the other two
    FMs are loaded in a separate Modal container the orbit cannot import.

## Solution Interface

```python
from typing import Literal
import jax

def search(
    substrate_name: Literal["lenia", "flow_lenia"],
    search_fm: Literal["clip_vitb32", "dinov2_vitb14", "siglip_vitb16"],
    prompt_bank_train: list[list[str]],   # 10 sequences × 5 prompts (training split)
    seed_pool_train: jax.Array,            # shape [16], disjoint from S_test
    budget: dict,                          # {"wall_clock_s": 1800}  hard 30-min cap
    rng: jax.Array,                        # PRNGKey
) -> dict:
    """
    Returns
    -------
    {
      "best_params":  jax.Array,    # [D] float32, single best candidate
      "archive":      jax.Array,    # [<=16, D] float32, diverse top-K (diagnostic only)
      "search_fm":    str,          # must equal the input search_fm (verified)
      "trace":        dict,         # {"generations": int, "best_per_gen": [G], ...}
    }

    Constraints
    -----------
    - Bit-identical determinism under fixed rng (XLA_FLAGS deterministic set by evaluator).
    - Must respect budget["wall_clock_s"]; evaluator enforces hard SIGKILL at cap.
    - May NOT import clip/dino/siglip modules other than `search_fm` (process-audited).
    - May NOT read prompt_bank_test, seed_pool_test, or any file under `eval/heldout/`.
    """
```

Orbits submit a `solution.py` exposing `search`. Scoring is done outside
`solution.py` by the frozen evaluator.

## Evaluation Procedure

1. Evaluator receives a `solution.py` path and a run_id. Instantiates the
   orbit in a Modal container with only `search_fm`'s FM module importable.
2. Calls `solution.search(...)` with `seed_pool_train` and
   `prompt_bank_train`; enforces `budget["wall_clock_s"] = 1800` SIGKILL.
3. On return, extracts `best_params`. Also records `archive` for diagnostics.
4. Scoring container: for each (s ∈ S_test, p ∈ P_test, f ∈ F):
   - Roll out `substrate(best_params, s)` for T=256 steps.
   - Sample 5 equally-spaced frames, render to RGB (Flow-Lenia: fixed colormap
     on mass channel + flow magnitude overlay; Lenia: grayscale→viridis).
   - Compute PG_eff per formula.
5. Compute LCF(f) for each f; compute METRIC = min_f LCF(f) − anchor.
6. Paired Wilcoxon test: per-seed PG_eff of θ vs. baseline θ*, over 16 test
   seeds, one-sided.
7. Output: `print(f"METRIC={value:.6f}")` and structured
   `METRIC_COMPONENTS={...}` with sub-scores and Wilcoxon p-value.

## Baselines

| Name | Expected LCF-robust@heldout (raw, pre-subtraction) | Source |
|------|----------------------------------------------------|--------|
| Trivial random params (single sample) | ~0.00 ± 0.01 | Lenia/Flow-Lenia random init |
| Random search (pop 64 × gens 200, pick best by train PG) | ~0.02 | 30-min budget-matched baseline |
| **ASAL Sep-CMA-ES (pinned)** | **~0.03–0.08** (to measure at init) | ASAL repo @ pinned SHA, CLIP-only search, Sep-CMA-ES from evosax, pop=16, σ=0.1, gens=200, PRNGKey(0) |
| SOTA aspirational | +0.05 over ASAL baseline (our target) | our campaign |

ASAL baseline is computed ONCE at campaign init on both train and test
splits, frozen in `research/eval/baseline/baseline_score.json`, and never
recomputed.

## Feasibility Assessment

- **Estimated eval time per run:**
  - `search()`: 30 min H100 (hard cap)
  - Held-out scoring: 16 seeds × 10 prompts × 3 FMs × 256-step rollout + 5-frame embed
    ≈ 480 embed ops × ~2s = ~16 min on H100
  - **Total per orbit submission: ~45 min** (well under 4 h)
- **Estimated eval cost per run:** ≈ 0.75 H100-hr per orbit. At ~$3/H100-hr
  ≈ $2–3 per orbit.
- **Determinism strategy:**
  - `XLA_FLAGS=--xla_gpu_deterministic_ops=true`
  - `jax.config.update("jax_default_prng_impl", "threefry")`
  - Pinned `jax==0.4.38`, `flax==0.10.2`, `transformers==4.47.1`, `evosax==0.1.6`
  - FM forward passes deterministic at float32
  - Rollout uses `jax.lax.scan` with explicit seeds
- **Infrastructure dependencies:**
  - Modal container image with jax[cuda12], flax, transformers, evosax, qdax,
    pyyaml
  - Modal secret `huggingface` mounting HF hub token (for CLIP/DINOv2/SigLIP
    weights; not an OpenRouter-style API key)
  - 1× H100 per orbit
  - Persistent Modal volume for cached FM weights (avoid re-download)
- **Circular eval risk:** none — FMs are frozen, never trained during campaign.
  Evaluator uses FMs, search may use `search_fm` but the two held-out FMs are
  process-isolated.

## Novelty Operationalization

The campaign's research novelty is captured in the permutation gap PG
(ordered-match signal) and the worst-of-FMs aggregation (measurement-robust
novelty). Every orbit must show `PG > 0` on held-out prompts/seeds/FMs to
score non-zero — this is the operational novelty bar.

An orbit that merely reproduces ASAL's softmax-target objective will score
LCF ≈ anchor on CLIP (permutation gap ~= 0 because ASAL does not explicitly
optimize ordering), so METRIC ≈ 0 and will fail the pass bar.

## Open Questions for Human

1. **Prompt authorship:** agreed that panel orchestrator drafts 20 life-cycle
   sequences (from biology canon — embryogenesis, metamorphosis,
   reproduction, population dynamics, etc.), human curates before baseline
   freeze. Block Phase 2 on this deliverable.
2. **SigLIP Flax loader:** HF does not ship `FlaxSiglipModel` as of 2026-04.
   Options: (a) port from big_vision (~1 engineering day);
   (b) PyTorch-in-Modal-subprocess with numpy tensor round-trip (simple, ~5%
   perf cost); (c) fall back to CLIP + DINOv2 (2 FMs, worst-of-2). Default:
   **(b)** unless (a) becomes trivial.
3. **Flow-Lenia rendering:** 2-channel state (mass, flow) rendered to RGB via
   `rgb = stack([mass, 0.5*flow_magnitude, 0.2*mass], axis=-1)` clipped to
   [0,1], then quantized. Fixed at eval-v1 freeze. Any objections?
4. **Budget overrun policy:** `search()` SIGKILL at 30 min. On SIGKILL,
   evaluator uses the last checkpointed `best_params` if available, else
   METRIC = 0.0. Solutions must checkpoint every ≥ 30 s.
5. **Agreement-τ promotion:** in campaign 1, Kendall-τ agreement between FMs
   is reported but not gating. Promote to a gate at orbit 10. Confirm?
